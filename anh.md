# Ghi nhớ nhanh trước khi kiểm tra

## Dễ ra chọn đáp án

- **damage** thường đi với: `the environment`, `the ozone layer`.
- **lead to** = dẫn đến.
- **solution to** = giải pháp cho.
- **negative effect on** = ảnh hưởng tiêu cực lên.
- **miss a flight** = lỡ chuyến bay.
- **delayed flight** = chuyến bay bị hoãn.
- **used to + V** là đúng.
- **used to V-ing** là sai trong nghĩa “đã từng”.

## Cặp dễ nhầm

| Từ | Loại từ / Nghĩa |
|---|---|
| affect | động từ: ảnh hưởng |
| effect | danh từ: ảnh hưởng / kết quả |
| permit | cho phép |
| prohibit / ban | cấm |
| promote | thúc đẩy |
| discourage | ngăn cản |
| native | bản địa |
| foreign | nước ngoài |
| simple | đơn giản |
| complicated / complex | phức tạp |
